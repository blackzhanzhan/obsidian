## 🚨 架构师防翻车预警 (Anti-Bug 雷达)
*(注：本模块记录了我在推演中成功避开，但大多数普通考生极易死在里面的考场陷阱)*

* **🛡️ 成功避坑 1：光密光疏的“绝对化”陷阱**
  * **考场高频死穴**：死记硬背“水是光密介质，空气是光疏介质”，遇到水和玻璃的界面时直接宕机。
  * **我的极客直觉**：引入“身高比较法”或“零势面参考系”！极其严厉地向学生强调，光疏与光密是**相对的**！没有对比，就没有密与疏。
* **🛡️ 成功避坑 2：折射角为 0 的几何直觉错觉**
  * **考场高频死穴**：推导临界角公式 $\sin C = 1/n$ 时，以为折射光消失就是角度为 0。
  * **我的极客直觉**：死死咬住“法线”！折射光线消失的瞬间，是它**掠过界面**的瞬间。它与法线的夹角达到了物理极限的 **90°**！利用 $\sin 90^\circ = 1$ 瞬间完成降维推演。

## 🛤️ 【Class A】Q12 局部微格教学库 (12分片段储备)
*(注：专打考场上最爱考的“如何引导推导临界角公式”片段)*

**🔹 Snippet 1：专考【物理极限状态的代数转化】**
* **师**：同学们，当入射角越来越大，折射光线会越来越贴近水面。当它恰好贴着水面消失的那个瞬间，我们把这个入射角叫“临界角 $C$”。请问，此时折射光线和法线的夹角（折射角）是多少度？
* **生A**：（看图）它贴着水面，法线是垂直的，所以是 90°！
* **师**：太棒了！现在，请大家祭出我们的折射定律：$n_1 \sin\theta_1 = n_2 \sin\theta_2$。光从折射率为 $n$ 的介质射向折射率为 1 的真空，请大家把刚才的极限角度代进去！
* **生B**：（在黑板上书写）$n \cdot \sin C = 1 \cdot \sin 90^\circ$。因为 $\sin 90^\circ$ 是 1，所以得出 $\sin C = \frac{1}{n}$！
* **师**：完美的推演！这就意味着，只要我们知道一种材料的折射率 $n$，就能极其精准地算出让它发生全反射的临界角 $C$！

## 🛤️ 【Class B】Q13 全局教案模板 (28分重装机甲)
*(注：涵盖纯逻辑推演与理论建构，控制在 500 字左右)*

### 一、 教学目标
1. **物理观念**：理解光疏介质和光密介质的相对性；掌握全反射的发生条件及临界角概念。
2. **科学探究与思维**：通过观察实验现象，归纳总结发生全反射的临界条件；能运用折射定律严密推导临界角公式 $\sin C = 1/n$。
3. **科学态度与责任**：了解光纤通信的原理，感受物理学改变人类社会通信方式的巨大力量（STS 教育）。

### 二、 教学重难点
* **重点**：全反射的发生条件（密 $\rightarrow$ 疏，且入射角 $\ge C$）；临界角公式推导。
* **难点**：理解光疏/光密介质的相对性；临界状态下折射角为 90° 的几何建构。

### 三、 教学过程

**环节一：创设情境，引发思考 (导入)**
* **活动**：教师演示“水流导光”实验（在塑料瓶侧面开孔放水，用激光笔从另一侧照射）。学生惊奇地发现光没有沿直线射出，而是“弯曲”地顺着水流导向了水槽。引入课题：全反射。

**环节二：师生互动，探究新知**
* **活动 1：概念扫盲（光密与光疏）。** 引入两种介质的折射率对比。通过提问“水是光密还是光疏”，强制学生建立“相对比较”的观念，绝杀绝对化错误。
* **活动 2：寻找临界点。** 利用激光演示器在半圆形玻璃砖中演示光从玻璃射向空气。逐渐增大入射角，学生观察到折射光变暗且靠近界面，最终消失。总结发生全反射的两个绝对条件。
* **活动 3：代数推演。** 抓住“折射角为 90°”这一极限状态，引导学生代入折射定律，自行推导出临界角公式 $\sin C = \frac{1}{n}$。

**环节三：生活应用，STS 升华**
* **活动**：揭秘导入中的“水流导光”本质。抛出诺贝尔物理学奖成果——光纤。提问：“为了把光锁在里面，内芯和外套谁的折射率必须更大？”学生根据全反射条件齐答：“内芯！”

### 四、 板书设计
**《全反射》**
一、 概念准备：光疏介质 与 光密介质（相对性！）
二、 全反射：
    1. 条件：①光密 $\rightarrow$ 光疏； ②入射角 $\ge$ 临界角 $C$
    2. 临界角公式：**$\sin C = \frac{1}{n}$** （此时折射角 $= 90^\circ$）
三、 应用：光纤通信（内芯折射率 $>$ 外套折射率）
