## 🚨 架构师防翻车预警 (Anti-Bug 雷达)
*(注：本模块记录了我在推演中成功避开，但大多数普通考生极易死在里面的考场陷阱)*

* **🛡️ 成功避坑 1：电压无法归零的“限流”陷阱**
  * **考场高频死穴**：画电路图时习惯性画成滑动变阻器限流式接法，导致电压根本无法从 0 开始调节，特性曲线缺失前半段。
  * **我的极客直觉**：果断采用**滑动变阻器分压式接法**。只要题目要求“电压从 0 开始连续变化”或“测量范围尽量大”，闭眼画分压电路，这是本节课的绝对生命线！
* **🛡️ 成功避坑 2：系统误差爆炸的“内接”陷阱（附底层防弹矩阵）**
  * **考场高频死穴**：随便接个电流表，导致测量出的电阻比真实值大得多。
  * **我的极客直觉**：精准指出必须用**电流表外接法**。因为小灯泡是典型的小电阻。
  * **🧠 核心记忆矩阵（八字真言：大内偏大，小外偏小）**：
    * **内接法** -> 电流表串联分压 -> 测量值偏大 -> 适用于大电阻测量。
    * **外接法** -> 电压表并联分流 -> 测量值偏小 -> 适用于小电阻测量。
* **🛡️ 成功避坑 3：线性思维定势的“温度”盲区**
  * **考场高频死穴**：以为所有导体的 $U-I$ 图像都是过原点的直线。
  * **我的极客直觉**：指出小灯泡 $U-I$ 图像是曲线的物理本质——随着电压和电流增加，灯丝发热导致**工作温度升高，金属的电阻率随之急剧增大**，图像斜率随之改变。

## 🛤️ 【Class B】Q13 全局教案模板 (28分重装机甲)
*(注：包含高阶电路图设计与核心数据处理，控制在 500 字左右)*

### 一、 教学目标
1. **物理观念**：知道小灯泡的伏安特性曲线是非线性的，理解温度对金属电阻率的影响规律。
2. **科学探究与思维**：能根据“电压从零开始连续变化”的要求独立设计**分压式**电路；能根据“小电阻”特性选择**外接法**；具备处理非线性实验数据、化解系统误差的能力。
3. **科学态度与责任**：在设计复杂电路和排查故障的过程中，养成严谨求实、安全操作的科学态度。

### 二、 教学重难点
* **重点**：描绘小灯泡的 $U-I$ 图像，理解其电阻随温度变化的物理本质。
* **难点**：滑动变阻器分压接法与电流表外接法的电路设计与实物连线论证。

### 三、 教学过程

**环节一：创设情境，引发思考 (导入)**
* **活动**：展示一个定值电阻和一个小灯泡。提问：初中我们测过定值电阻的 $U-I$ 图像是直线，如果换成小灯泡，它还会是一条直线吗？引出探究课题。

**环节二：师生互动，探究新知**
* **活动 1：攻克核心电路设计。** - 抛出任务 1：要求电压必须从 0 开始。引导学生淘汰限流式，确立**分压式**接法。
  - 抛出任务 2：已知小灯泡电阻仅有几欧姆。引导学生复习“大内偏大，小外偏小”，确立**电流表外接法**。
  - 师生共同在黑板上画出终极电路图，并强调开关闭合前，滑片必须置于分压为 0 的最左端以保护电路。
  - **【设计意图】**：通过连环设问，将复杂的电路设计拆解为两个核心物理决策，锻炼高阶逻辑思维。
* **活动 2：分组实验，作图分析。** 调节滑片，记录多组 $(U, I)$ 数据，在坐标纸上描点平滑连线。
* **活动 3：探寻曲线背后的真相。** 面对非线性曲线，引导学生分析灯泡发热现象，得出电阻率随温度升高而增大的物理本质。

### 四、 板书设计
**《描绘小灯泡的伏安特性曲线》**
一、 核心电路设计原则：
    1. 电压从零起步 $\rightarrow$ **分压式**接法
    2. 小灯泡（小电阻） $\rightarrow$ 电流表**外接法**（小外偏小）
二、 数据图像（草图展示 $U-I$ 向上弯曲的曲线）
三、 物理本质：金属电阻率随**温度**升高而增大。
